# Array Programs in Java

This repository contains Java programs related to arrays, covering basic to intermediate concepts commonly used in Data Structures and Algorithms (DSA).

## 📚 Topics Covered

- Array Creation and Initialization
- Input and Output Operations
- Traversing Arrays
- Finding Maximum and Minimum Elements
- Sum and Average of Array Elements
- Searching Algorithms
  - Linear Search
  - Binary Search
- Sorting Algorithms
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
- Array Reversal
- Second Largest Element
- Duplicate Element Detection
- Frequency Counting
- Multi-Dimensional Arrays
- Practice Questions

## 🚀 Getting Started

### Prerequisites

- Java JDK installed
- Any Java IDE or Code Editor

### Run a Program

1. Clone the repository:

```bash
git clone https://github.com/zedsparky/array.git
```

2. Navigate to the project directory:

```bash
cd array
```

3. Compile the Java file:

```bash
javac FileName.java
```

4. Run the program:

```bash
java FileName
```
